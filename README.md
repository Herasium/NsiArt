## Pour installer le jeu:

Clone le projet

```bash
  git clone https://github.com/Herasium/NsiArt
```

Ouvre le dossier du projet

```bash
  cd NsiArt
```

Installe les dependances (numpy)

```bash
  pip install -r requirements.txt
```
ou 
```bash
  pip install numpy
```

Ouvre le jeu

```bash
  python __main__.py
```
